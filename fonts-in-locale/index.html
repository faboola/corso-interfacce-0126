Hosting dei fonts in locale

Inn alternativa a Google Fonts, posso ospitare i font direttamente sul mio server e includerli nel mio progetto tramite CSS.

Questa pagina ad esempio usa il font "IBM Plex Sans" in locale.

Come si fa?

Possiamo aiutarci con un tool online come https://gwfh.mranftl.com/fonts

  1. Scelgo i fonts
  2. Copio il CSS generato dall'helper
  3. Scarico i files dei fonts e li metto nella cartella del mio progetto
  4. Incollo il CSS nel mio file CSS principale
  5. Uso i font nel mio CSS
